# Changelog

All notable changes to FinSight Pro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of FinSight Pro
- PDF, CSV, and Excel file upload support
- Automatic transaction classification (income/expense)
- Executive financial dashboard with key metrics
- Monthly and yearly trend visualizations
- Category-wise spending analysis
- Best/Worst month insights
- AI-powered anomaly detection using Isolation Forest
- K-Means clustering for transaction and monthly patterns
- Month-to-month comparison tool
- Year-over-year comparison feature
- Global year filter for all views
- CSV and Excel export capabilities
- Interactive Plotly visualizations
- Professional glassmorphism UI design
- Transaction table view with sorting and filtering
- Detailed setup and usage documentation

### Features
- Smart file extraction with automatic column detection
- Multiple date format support
- Configurable anomaly detection sensitivity
- Transaction-level and monthly clustering modes
- Category drilldown analysis
- Export options for all views
